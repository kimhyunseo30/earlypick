# EarlyPick

온라인 관심 신호와 시장 데이터를 비교 분석하여 식품 트렌드 후보를 조기에 탐색하고, 진입 타이밍 판단을 돕는 데이터 기반 의사결정 서비스입니다.

## 프로젝트 배경

식품 트렌드는 이미 유행한 이후 발견되는 경우가 많아, 상품 기획 및 마케팅 의사결정 시점이 늦어지는 문제가 있습니다.

EarlyPick은 네이버 쇼핑 인사이트 순위 데이터와 검색·쇼핑 반응 데이터를 활용하여 주목할 만한 식품 후보를 조기에 탐색하는 것을 목표로 했습니다.

## 주요 기능

- Top20 / Top200 순위 데이터 수집 및 분석
- 상품명 표준화
- 상품 유형 분류
- 검색량 및 쇼핑 반응 비교
- 반복 등장 상품 탐색
- 규칙 기반 트렌드 후보 판단
- AI 인사이트 문구 자동 생성 실험

- 상품명 표준화 로직 설계

- 규칙 기반 판단 기준 설계
- 주요 화면 및 인사이트 기능 기획

## 분석 결과

후보 상품 데이터를 분석한 결과, 21~50위 구간과 과거 Top20 진입 이력이 있는 상품이 상대적으로 높은 Top20 진입 가능성을 보이는 패턴을 확인했습니다.

이를 통해 단순 인기 순위 확인이 아닌, 트렌드 후보를 조기에 탐색하기 위한 데이터 기반 판단 기준을 설계할 수 있었습니다.

## 향후 계획

현재는 규칙 기반 분석 MVP 단계이며, 향후 데이터 누적을 통해 AI 기반 트렌드 예측 및 인사이트 자동 생성 기능으로 확장할 예정입니다.
